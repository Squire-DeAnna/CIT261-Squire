Image Size - HTML5 and CSS code


HTML5
  1. Small: < img src = “(location of the image)” Class = “small” alt= “(title)/>
  2. Medium: < img src = “(location of the image)” Class = “medium” alt= “(title)/>
  3. Large: < img src = “(location of the image)” Class = “large” alt= “(title)/>

CSS
  1. Small: img.small { width: 100px;height: 100px;}
  2. Medium: img.medium { width: 250px;height: 250px;}
  3. Large: img.large { width: 500px;height: 500px;}
