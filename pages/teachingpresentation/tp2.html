Image Size - Why CSS?


  1. CSS helps to regulate the size of images on the website so that they are all equal. This keeps your website looking clean and crisp.
  2. It eliminates that need for remembering the exact size for images to use later
  3. Allows you to adjust all the images at once with just changing the CSS code.
